# Tmap 🌐
A professional Python CLI wrapper for Nmap reconnaissance, created by An1En.

Tmap automates the discovery phase of penetration testing, allowing you to cast a wide net for open ports and then automatically execute deep service enumeration on the discovered targets.

## 🚀 Global Installation (Linux)

To install Tmap on any Debian/Ubuntu/Kali system, run the following commands in your terminal:

```bash
# 1. Clone the repository
git clone https://github.com/An1En/Tmap.git

# 2. Navigate into the directory
cd Tmap

# 3. Run the installer
sudo ./install.sh

#4.excute
tmap-scan
